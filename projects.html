Projects

COVID-19 Visualizations with Data Science and Machine Learning

Visualizations of the novel coronavirus with Python. This was my first step into data science and I really learned a lot through libraries such as Pandas, GeoPandas, NumPy, and MatPlotLib. I primarily used data from NYTimes to analyze US cases and data from JHU CSSE to analyze international cases.

Gitlet

A smaller version control system replica of Git that is written entirely in Java. This was for a school project in UC Berkeley's CS 61BL, Data Structures and Programming Methodology, where we exercised the use of data structures and software engineering skills to design and implement a version control system.

BearMaps

Implemented a small scale Google Map replica of UC Berkeley's map written in Java. This was for a school project in UC Berkeley's CS 61BL, Data Structures and Programming Methodology. We used data structures and algorithms to perform map engine functions, such as rasterisation, A* algorithms for route finding, tries for search queries, and k-d trees for finding nearest pin points. Spec is availiable here as well as an app
