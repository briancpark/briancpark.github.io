Scheme Interpreter

This is a project developed in Python3. This was the final project for UC Berkeley's CS 61A, which is an introductory course to programming for Computer Science majors. The full spec written for the project can be found here. Simply run the command python3 scheme.py to run the interpreter. It will run the Scheme REPL in the terminal, complete with all Scheme functions and properties. Even showing you bug errors.
