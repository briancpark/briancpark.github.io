Mandelbrot Movie

This is a project developed in C (C99 Programming Language Standard). The main function behind this project is motivated by the Mandelbrot function, M(Z, C) = Z^2 + C. Where Z is the center and C is a constant and Z is an iteration variable. We are interested in how many times Z can iterate, and we set a maximum iteration value because some functions can iterate indefiitely. M and C can both be imaginary numbers. The program also deals with how colors are represented when plotting the Mandelbrot function on a canvas. We use the P6 color pallete where each unique color corresponds to a unique iteration value that is moduloed to a set number.
